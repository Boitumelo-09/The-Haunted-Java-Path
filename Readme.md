## 🕯️ PHASE 0 — THE RITUAL OF ENTRY (Days 1–3)

**Purpose:** Stop resisting. Accept the JVM.

### You must understand:

* What the JVM actually is
* `.java → .class → JVM`
* Why Java has **no `.h` / `.cpp`**
* Garbage Collection (conceptual, not implementation)

### Tools (non-negotiable):

* JDK 17+
* IntelliJ IDEA (Community)
* Git

**Outcome:**

> You stop asking *why Java* and start asking *how Java thinks*.

---

## 🕯️ PHASE 1 — THE SYNTAX GRAVEYARD (Week 1)

**Purpose:** Kill beginner fear immediately.

### Learn ONLY:

* Variables & data types
* Conditionals
* Loops
* Methods
* Arrays
* `String` (Java’s most used weapon)

### Daily rule:

> Every concept → one tiny program

Examples:

* Number checker
* Simple calculator
* Student average calculator

**Outcome:**

> Syntax stops being noise.

---

## 🕯️ PHASE 2 — OBJECTS & CURSES (Week 2)

**Purpose:** You stop scripting. You start designing.

### Master:

* Classes & Objects
* Constructors
* Encapsulation
* `this`
* `static`

### Project:

**Student Management System (CLI)**

* Add student
* View students
* Search by ID

**Outcome:**

> You think in objects, not instructions.

---

## 🕯️ PHASE 3 — INHERITANCE & POLYMORPHISM (Week 3)

**Purpose:** Runtime truth is revealed here.

### Learn deeply:

* Inheritance
* Method overriding
* `super`
* `final`
* Polymorphism
* `Object` class (`toString`, `equals`)

### Project:

**Bank Account System**

* Base `Account`
* `SavingsAccount`, `CurrentAccount`
* Override behavior

**Outcome:**

> You understand behavior, not just structure.

---

## 🕯️ PHASE 4 — INTERFACES & ABSTRACTION (Week 4)

**Purpose:** Java’s true strength.

### Master:

* Interfaces
* Abstract classes
* When to use which
* Dependency thinking

### Project:

**Payment System**

* `PaymentMethod` interface
* Implement `Card`, `Cash`, `MobileMoney`

**Outcome:**

> Your code becomes replaceable, testable, trusted.

---

## 🕯️ PHASE 5 — EXCEPTIONS & FILES (Week 5)

### Project:

**File-Based Login System**

**Outcome:**

> Your programs stop crashing like amateurs.
